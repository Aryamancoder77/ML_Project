# 🧠 Fake News Detection using BERT

This project uses a BERT-based transformer model to classify news as **Real or Fake**.

## 🚀 Features
- NLP using BERT
- High accuracy classification
- Flask API for deployment

## 📂 Dataset
Use Kaggle Fake News dataset and place it in `data/dataset.csv`
